What the system does

A two-stage pipeline designed for fast, interpretable bone anomaly detection from standard X-ray input.

🧠
CNN Classification
A deep convolutional network trained on the MURA dataset classifies the X-ray as Normal or Abnormal with a confidence score.
🔍
YOLOv8 Localization
When abnormality is detected, YOLOv8 draws bounding boxes around suspected fracture regions with sub-pixel accuracy.
🌡
Thermal Heatmap
Each detected region is overlaid with a COLORMAP_JET thermal visualization — hotter tones indicate higher anomaly density.
📊
Severity Grading
Confidence scores are mapped to a three-tier severity scale: Mild (<40%), Moderate (40–70%), and Severe (>70%).
⚡
Real-time Processing
Images are resized, normalized, and processed through both models in a single forward pass — results in seconds.
🖼
Side-by-side Viewer
Compare the original X-ray against the thermal output in a dual-panel modal window with annotated result data.
